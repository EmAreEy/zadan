# Zadan

بررسی احتمال درگیری نظامی با ایران به وسیله پایش Polymarket و فرستادن هشدار در تلگرام.

---

## معرفی پروژه

این پروژه داده های پیشبینی حمله نظامی به ایران رو از Polymarket به صورت دوره ای دریافت میکنه و با مقایسه داده های جدید با داده های اجرای قبلی میزان تغییرات و افزایش تنش رو بررسی میکند و در صورت عبور از ترشهولد های موردنظر یک آلارم در تلگرام میفرستد.

---

## منبع داده

* Polymarket Gamma API
* بازار:
  `israel-strikes-iran-by-february-28-2026`

داده‌های استخراج‌شده:

* احتمال YES
* احتمال NO
* حجم معاملات

---

## منطق هشدار

ربات تغییرات قیمت و حجم معاملات را بررسی کرده و بر اساس ترکیبی از این دو، وضعیت بازار را در یکی از حالت‌های زیر دسته‌بندی می‌کند:

* `nazadan` — وضعیت عادی
* `yellow` — هشدار متوسط
* `red` — هشدار جدی

---

## نحوه اجرا

### اجرای خودکار

ربات به‌صورت خودکار هر ۱۵ دقیقه توسط GitHub Actions اجرا می‌شود.

مراحل اجرا:

1. دریافت داده از API
2. مقایسه با داده‌ی قبلی
3. بررسی شرایط هشدار
4. ارسال پیام تلگرام
5. به‌روزرسانی داده ذخیره‌شده

---
